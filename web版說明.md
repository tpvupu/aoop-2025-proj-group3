# 🌐 網頁版說明

## 新功能：HTML5 網頁版

現在你可以直接在瀏覽器中玩這個遊戲了！無需安裝 Python 或任何依賴。

### 🚀 快速開始

```bash
cd web
./start.sh
```

然後在瀏覽器訪問：**http://localhost:8000**

### ✨ 網頁版特色

- 🌐 **跨平台** - Windows、Mac、Linux 通用
- 📱 **響應式** - 適配不同螢幕尺寸
- 🚀 **零安裝** - 瀏覽器直接運行
- 🔗 **易分享** - 一個連結即可分享
- ☁️ **雲端部署** - 可部署到 GitHub Pages、Netlify 等

## Python 版 vs 網頁版

| 特性 | Python 版 | 網頁版 |
|------|----------|--------|
| 安裝 | 需要 Python + Pygame | 無需安裝 |
| 運行 | 執行 main.py | 瀏覽器打開 |
| 分享 | 發送執行檔 | 發送連結 |
| 平台 | 需針對平台打包 | 天然跨平台 |
| 更新 | 需重新下載 | 自動更新 |
| 圖形 | GIF 動畫 | Canvas 渲染 |
| 部署 | 本地執行 | 雲端託管 |

**兩個版本都完全可玩！** 選擇你喜歡的方式 🎮
