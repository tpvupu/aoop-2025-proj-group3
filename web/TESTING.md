# 網頁版快速測試指南

## 🚀 立即測試

### 方法 1：使用啟動腳本（推薦）

```bash
cd web
./start.sh
```

然後在瀏覽器打開：**http://localhost:8000**

### 方法 2：手動啟動伺服器

**使用 Python 3：**
```bash
cd web
python3 -m http.server 8000
```

**使用 Python 2：**
```bash
cd web
python -m SimpleHTTPServer 8000
```

**使用 Node.js：**
```bash
cd web
npx http-server -p 8000
```

### 方法 3：使用 VS Code

1. 安裝擴充套件：**Live Server**
2. 右鍵點擊 `web/index.html`
3. 選擇 **"Open with Live Server"**

## 🐛 除錯技巧

### 打開瀏覽器開發者工具

- **Windows/Linux**: 按 `F12` 或 `Ctrl + Shift + I`
- **Mac**: 按 `Cmd + Option + I`

### 查看錯誤訊息

在 **Console** 標籤中查看是否有紅色錯誤訊息。

### 常見問題

#### 1. 頁面空白
- 檢查是否使用 HTTP 伺服器（不要直接打開 HTML 文件）
- 查看 Console 是否有 CORS 錯誤

#### 2. 場景無法切換
- 確認所有場景文件都已載入
- 檢查 `index.html` 中的 `<script>` 標籤順序

#### 3. 按鈕無反應
- 確認 Phaser 正確載入
- 查看 Console 的 JavaScript 錯誤
