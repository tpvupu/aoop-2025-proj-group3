## 🚀 快速開始

### 本地運行
```bash
cd web
./start.sh
# 或
python3 -m http.server 8000
```

然後訪問：http://localhost:8000

### 部署到 GitHub Pages
```bash
cd web
git init
git add .
git commit -m "Web version"
git remote add origin YOUR_REPO_URL
git push -u origin main
```

在 GitHub 設定中啟用 Pages 即可。

## 🎓 技術亮點

### 1. 完整的場景系統
使用 Phaser 3 的場景管理器，每個場景獨立開發，耦合度低。

### 2. 全局狀態管理
通過 `GameState` 單例管理玩家數據，跨場景共享。

### 3. 面向對象設計
`Character` 類完整封裝角色邏輯，易於維護和擴展。

### 4. 事件驅動架構
利用 Phaser 的事件系統，實現高效的用戶互動。

### 5. 響應式 UI
所有 UI 元素自適應螢幕大小，支持不同設備。
