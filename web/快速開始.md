# 🎮 今天的我也想耍廢 - 網頁版

## ⚡ 3 分鐘快速開始

### 第一步：啟動遊戲

```bash
cd web
./start.sh
```

### 第二步：打開瀏覽器

訪問：**http://localhost:8000**

### 第三步：開始遊戲！

1. 點擊「開始遊戲」
2. 選擇你喜歡的角色
3. 開始你的大學生活！

## 🎨 下一步（可選）

### 整合資源
```bash
./setup_assets.sh  # 創建資源目錄
```

然後：
1. 複製圖片到 `assets/images/`
2. 複製音效到 `assets/sounds/`
3. 更新 `PreloadScene.js`

### 部署到雲端
```bash
# 查看部署指南
cat DEPLOYMENT.md
```

支持平台：
- GitHub Pages

- Firebase Hosting
